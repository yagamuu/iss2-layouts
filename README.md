# INDIE Speedrun Summit 2 Layouts
INDIE Speedrun Summit 2というイベント用に作成したNodeCGのbundleです。  
This is the [NodeCG](http://github.com/nodecg/nodecg) bundle used for the streaming overlay in INDIE Speedrun Summit 2.  

## Usage
```
npm install
docker compose up
```
